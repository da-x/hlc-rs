# hlc-rs
A [Hybrid Logical Clock](http://muratbuffalo.blogspot.de/2014/07/hybrid-logical-clocks.html) for [Rust](http://rust-lang.org).

The latest documentation is always [here](https://tschottdorf.github.io/hlc-rs/doc/hlc/).
